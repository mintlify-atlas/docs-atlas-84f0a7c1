## Quick Start

Get up and running with Villa Buena E-Commerce in minutes.

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install dependencies
    
    <CodeGroup>
    ```bash npm
    npm install
    ```
    
    ```bash yarn
    yarn install
    ```
    
    ```bash pnpm
    pnpm install
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Configure Environment">
    Set up your environment variables for API integration
    
    ```bash .env
    VITE_API_URL=https://api.example.com
    ```
  </Step>
  
  <Step title="Start Development Server">
    Launch the development server
    
    ```bash
    npm run dev
    ```
  </Step>
</Steps>
